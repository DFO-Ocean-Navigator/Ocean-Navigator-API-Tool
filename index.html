API Documentation

How To:

The Ocean Navigator uses 2 different methods to receive API queries.
Both methods require the query to be prepended with "http://navigator.oceansdata.ca/"
The first method uses Query Parameters, and the second uses Template Parameters

(1) Query Parameters
The query method uses a base query followed by the input parameters
To indicate the start of variable input, a ? is used.
Similarly, to include more than one variable in the query, seperate each variable with a &
Variables included at the end of a query can be defined using
?variable= ...

Example: 
http://navigator.oceansdata.ca/api/variables/?dataset=giops_day&3d_only


?          : beginning of parameter input
dataset    : parameter with assigned value
=giops_day : assigning value to dataset
&          : indicates next parameter
3d_only    : standalone parameter (no value)
When scripting, longer queries should be written in JSON and converted to encodedURI format

(2) Template Parameters
Template based parameters differ from query based parameters in various ways:
- Parameter Order Matters
- Parameter values are separated by '/'
-Input paramater types must match expected parameter types
This means you cannot provide an iteger if the query is expecting a string.
The expectations for each query are outlined in the documentation below.

Example: 
http://navigator.oceansdata.ca/api/class4/forecasts/class4_20180531_GIOPS_CONCEPTS_2.3_profile


format: /api/class4/<string:q>/<string:class4_id>/


/api/class4/       : Base
<string:q>         : 1st Parameter Value
<string:class4_id> : 2nd Parameter Value

There are multiple ways to construct an API request in a script. 2 examples are given below.
**Both examples will print 4 lists displaying the depths available for each of the variables specified**

(1) String Replacement
In this example, the 3 curly brace sets "{}" will be replaced by the 3 values given in .format()
These values can be assigned, incremented, or modified to make multiple automated API requests. 

import urllib

base_url = "http://navigator.oceansdata.ca/api/depth/"
dataset = giops_day
variables = [votemper, vosaline, vozocrtx, vomecrty]

for variable in variables:

In this example, the variables are incorporated directly in place, and are concatenated together in the order they appear
The text given in quotations "..." are hardcoded strings which will become part of the query 

import urllib

dataset = giops_day
variables = [votemper, vosaline, vozocrtx, vomecrty]

for variable in variables:
	url = "http://navigator.oceansdata.ca/api/depth/?dataset=" + dataset + "&variables=" + variable

API Format: /api/variables/?dataset=<dataset_id>&3d_only=<string:old_dataset>&vectors_only=<string:old_dataset>&vectors=<string:old_dataset>

                        **Only use variables required for your specific request**

                        dataset_i      : Dataset to extract data - Can be found using /api/datasets
                        3d_only      : Boolean Value; When True, only variables with depth will be shown
                        vectors_only : Boolean Value; When True, ONLY variables with magnitude will be shown 
                        vectors      : Boolean Value; When True, magnitude components will be included

                        **Boolean: True / False**
